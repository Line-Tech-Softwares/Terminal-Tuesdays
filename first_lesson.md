## üíª Windows Terminal Essentials (15 Commands)

| Command | Meaning / Example |
|---------|-----------------|
| `dir` | Lists all files and folders in the current directory |
| `cd` | Change directory. Example: `cd Documents` moves you into the Documents folder |
| `mkdir` | Create a new folder. Example: `mkdir Projects` |
| `del` | Delete a file. Example: `del test.txt`. Be careful! |
| `ipconfig` | Shows network information like IP address and connection status |
| `cls` | Clears the terminal screen |
| `copy` | Copy files from one location to another. Example: `copy file.txt D:\Backup` |
| `move` | Move or rename files. Example: `move file.txt D:\Documents` |
| `ren` | Rename a file. Example: `ren oldname.txt newname.txt` |
| `type` | View the contents of a file. Example: `type notes.txt` |
| `tree` | Display the folder structure of a directory as a tree |
| `ping` | Test network connectivity. Example: `ping google.com` |
| `tasklist` | List all running processes |
| `taskkill` | Kill a process by name or PID. Example: `taskkill /IM notepad.exe /F` |
| `systeminfo` | Displays detailed information about your computer system |

## üêß Linux Terminal Essentials (Parrot OS) (15 Commands)

| Command | Meaning / Example |
|---------|-----------------|
| `ls` | Lists all files and folders in the current directory |
| `cd` | Change directory. Example: `cd Documents` |
| `mkdir` | Create a new folder. Example: `mkdir Projects` |
| `rm` | Delete a file or folder. Example: `rm test.txt`. Be careful! |
| `sudo apt update && sudo apt upgrade` | Updates your system and all packages |
| `pwd` | Prints the current working directory |
| `cp` | Copy files or folders. Example: `cp file.txt ~/Documents` |
| `mv` | Move or rename files/folders. Example: `mv file.txt ~/Documents` |
| `nano` | Open a file in the nano text editor. Example: `nano notes.txt` |
| `cat` | Display the contents of a file. Example: `cat notes.txt` |
| `touch` | Create a new empty file. Example: `touch newfile.txt` |
| `chmod` | Change file permissions. Example: `chmod +x script.sh` |
| `chown` | Change file ownership. Example: `sudo chown user:user file.txt` |
| `top` | Show running processes and system resource usage |
| `ping` | Test network connectivity. Example: `ping google.com` |
